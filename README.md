# ph-tin

[![npm version](https://img.shields.io/npm/v/ph-tin.svg)](https://www.npmjs.com/package/ph-tin)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Philippine TIN (Tax Identification Number) validation and formatting

## Installation

```bash
npm install ph-tin
```

## Functions

- `formatTIN` — Format TIN to XXX-XXX-XXX-XXX
- `validateTIN` — Validate TIN format
- `generateRandomTIN` — Generate random TIN for testing
